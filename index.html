Romania Energy Analysis

An exploratory data analysis of hourly electricity consumption and production data from Romania

Data & Methodology

How this analysis was conducted, from raw hourly data to insights

Data Coverage: 62,810 hourly records from 2019–2025 (with partial 2026). See Data Quality section for caveats.

Data Source

Hourly electricity consumption and production data from Romania's national grid. Covers hourly observations with detailed breakdown by energy source (Nuclear, Wind, Hydroelectric, Solar, Coal, Oil & Gas, Biomass).

Tools & Libraries

  • Python for data processing and analysis
  • Pandas and NumPy for data wrangling
  • Plotly for interactive visualizations
  • Statistical aggregation and time-series analysis

Process

  • Hourly data parsing and time-series aggregation
  • Data cleaning & feature engineering
  • Peak load and renewable production analysis
  • Energy source breakdown and seasonality
  • Import/export and self-sufficiency metrics
